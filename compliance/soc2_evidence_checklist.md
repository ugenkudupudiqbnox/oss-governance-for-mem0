# SOC 2 Evidence Checklist

This checklist helps teams prepare audit evidence using Governance Pack for Mem0.

## Identity & Access
- [ ] IAM configuration screenshots
- [ ] Role definitions (admin, agent, auditor)
- [ ] MFA configuration evidence

## Authorization
- [ ] Policy-as-code files (version controlled)
- [ ] Deny-by-default rules documented
- [ ] Tenant isolation policies

## Audit Logging
- [ ] Audit log schema
- [ ] Sample audit records
- [ ] Evidence of append-only storage

## Monitoring & Alerts
- [ ] Log aggregation dashboards
- [ ] Alert configuration screenshots

## Change Management
- [ ] Git history of policy changes
- [ ] Review and approval records

## Incident Response
- [ ] Log search demonstrating incident investigation
